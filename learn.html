Setting up your HTML/CSS file (Made by me)

  1. Open your text editor, could be almost any editor.
  2. Create a new file and name it index.html
  3. Start with typing the doctypeHTML tag. All tags start with < and end with > and the closing tag (if needed) has a / after the opening <.
  4. Next you need the html tag and closing html tag. Make space between them with enter.
  5. Do the same with head and body, but don't put them inside one another.
  6. Set up a stylesheet(called style.css or layout.css) and link stylesheet to index.html using this code in the html:
    <link rel="stylesheet" href="style.css" type="text/css" charset="utf-8">
